    写博客的初衷

    1. 任何知识，如果一上来就是讲数学原理，而不会使用，那么会劝退我们新手
    2. 数学的美的，是描绘自然最好的语言，在了解知识后，应该回归到数学，享受这个过程
    3. 博客的主要功能就是分享知识，通过分享知识，自己才会认真去学习这个知识，才能更好提高自己
    4. 本来想设置一个评论系统的，因为只有大家提问，才能检验自己是否真正明白了这个知识

    软件推荐

    学习软件：

    1. 有道云笔记

    我所有的笔记都是用的有道云笔记进行记录的，其同步功能非常好用，建议大家去学习markdown基础语法，同时，有道云的搜索功能非常强大，当我在实际编程时，我对什么知识点忘记了，但是只有大概的印象，就可以用有道云的搜索功能进行查阅

    1. GitHub

    这个网站时强大的学习网站，里面有大量的开源代码和众多的python和机器学习教程，向大佬们学习编程知识

    好书推荐

    python教程

    视频教程推荐：小甲鱼的python，b站上有视频教程，详细介绍了python的所有基础知识，初学用这个视频入门很不错，同样他的网站配备了大量的课后习题，每一个知识点都有对应的习题，写完后可以对知识点进行巩固

    通信原理

    可以淘宝搜索：《大话通信》，和《深入浅出通信原理》，我觉得知识不应该从数学开始，而应该先培养兴趣，阅读这两本书籍，可以帮忙扫清通信的基础知识，这样阅读其他书籍会相对于简单
