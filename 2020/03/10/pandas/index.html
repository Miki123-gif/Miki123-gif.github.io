pandas和numpy的区别

  • numpy一般是处理数值型数据
  • pandas则可以处理c语言中结构体的数据，里面可以混合字符串和数值

Series创建一维带标签的数组

  • 语法：Series([list])
print(pd.Series([1,2,3,np.nan,8]))

0    1.0
1    2.0
2    3.0
3    NaN
4    8.0
dtype: float64
  • np.nan means NaN
  • Series开头一定要大写
  • 默认存储的形式是float

  • pandas会把字典的所有键生成列的标签

处理丢失的数据

t = pd.DataFrame([[1,2,3,4],[2,3,4,5]],columns=list('abcd'))
t.iloc[0,1] = np.nan
print(t)
print('---------')
u = t.dropna(axis=0,how='any')#how={'any','all'}
print(u)

   a    b  c  d
0  1  NaN  3  4
1  2  3.0  4  5
---------
   a    b  c  d
1  2  3.0  4  5

  • axis=0表示处理行的数据，=1表示处理列，默认是处理行

  • 当how=‘any’时，则只要行数据中有NaN，则会删除这行，all是必须所有数据都为NaN才会删除

  • 注意不会对原先的数组造成影响

t = pd.DataFrame([[1,2,3,4],[2,3,4,5]],columns=list('abcd'))
t.iloc[0,1] = np.nan
print(t)
print('---------')
u = t.fillna(value=0)
print(u)

0  1  NaN  3  4
1  2  3.0  4  5
---------

0  1  0.0  3  4
1  2  3.0  4  5
