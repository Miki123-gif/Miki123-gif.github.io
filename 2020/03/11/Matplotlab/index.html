基本绘图

  • pycham中数据没法放大问题

使用pycharm时我们经常会用matplotlib库画图并show出来，但在pycharm默认的设置中，其结果（如下图所示）是在pycharm的右侧展示，而且不能放大图片的任意位置，甚至鼠标滚轮放大后只能用滑动条去移动放大的图片。因此我们需要更改设置，设置路径：file -> settings -> PythonScientific -> 将Show plots in tool windo 的✔去掉

  • 无法显示中文问题，加入这两行代码即可
plt.rcParams['font.sans-serif'] = ['SimHei']  # 步骤一（替换sans-serif字体）
plt.rcParams['axes.unicode_minus'] = False  # 步骤二（解决坐标轴负数的负号显示问题）

import matplotlib.pyplot as plt
import numpy as np

x = np.linspace(-1,1,50)
y = 2*x + 1
plt.plot(x, y)# 要画出来
plt.show()# 画出来还要显示出来

绘制两个函数图像，分别两个图

x = np.linspace(-1,1,50)
y1 = 2*x + 1
y2 = x**2

plt.figure()
plt.plot(x, y1)

plt.figure(num=3,figsize=(8, 5))
plt.plot(x, y2)

plt.show()

  • 每个figure对应一个位置，结果是显示两个图片窗口

  • 两个默认的参数num=,figsize=()，可以设置显示的窗口号，和图片的长宽

在一个图中画两个函数，并设置颜色

x = np.linspace(-1,1,50)
y1 = 2*x + 1
y2 = x**2

plt.figure()
plt.plot(x, y1)
plt.plot(x, y2, color='red', linewidth=1.0, linestyle='--')

plt.show()

  • 可以设置曲线的颜色等参数color='red', linewidth=1.0, linestyle='--'

  • 要将两个曲线画在同一个窗口中，要记得只设置以一个figure()

设置坐标轴取值范围(限制显示范围)

  • xlim(x limit),ylim
plt.figure()
plt.plot(x, y1)
plt.plot(x, y2, color='red', linewidth=1.0, linestyle='--')
plt.xlim(-0.5, 0.5)
plt.ylim(-1, 1)


plt.show()
  • 注意一定要在figure和show之间设置xlim

添加x轴或y轴的描述

plt.figure()
plt.plot(x, y1)
plt.xlabel('时间')
plt.ylabel('number')
plt.show()

更换x,y轴的数据显示形式

  • xticks(),yticks()

  • 使用场景，给别人看时一目了然

plt.figure()
plt.plot(x, y1)
plt.xlabel('时间')
plt.ylabel('number')
plt.yticks(ticks=[-1,-0.5,0.5,1],
           labels=['非常差','差','好','非常好']
           )
plt.show()

plt.figure()
plt.plot(x, y1)
plt.xlabel('时间')
plt.ylabel('number')
plt.yticks(ticks=[-1,-0.5,0.5,1],
           labels=[r'$really\ bad$',r'$bad$',r'$normal$',r'$really\ good$']
           )
plt.show()
  • r表示正则表达，两个美元符号表示以数学格式显示，更美观，空格要转义\

制作图例

  • 利用label属性，和legend方法
plt.figure()
plt.plot(x, y1, label='y1')
plt.plot(x, y2, label='y2')
plt.legend()
plt.show()

散点图绘制

n = 1024
x = np.random.normal(0, 1, n)
y = np.random.normal(0, 1, n)
t = np.arctan2(y, x)# 为了颜色好看
plt.scatter(x, y, s=20, c=t, alpha=0.5)# size color
plt.xticks(())# 不显示坐标
plt.yticks(())
plt.show()

x = np.arange(10)
y = x**2
plt.scatter(x, y)# 注意这里是先x后y
plt.show()
