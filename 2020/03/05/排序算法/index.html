排序算法的动画在这里，概念讲的很透彻

由于上面的排序代码是java写的，所以这里我会用python来实现一遍

什么是内部排序，什么是外部排序

  • 内部排序指的是在内存 里面进行的排序
  • 外部排序是当数据很大，内存存放不了的时候，需要用到外存进行数据排序

什么是排序算法的稳定性

通俗地说就是假如数据中有很多同样大小的数据，在排序中，如果这些数据的相对次序不会发生变化，则称这个排序算法是稳定的

相对次序是指排序前，你在我左边，排序后，你依旧在我左边

1. 冒泡排序

算法步骤（假设从小到大排序）

  1. 比较第一个和第二个数，如果第一个大于第二个，则交换两个数据位置
  2. 再比较第二个和第三个，重复交换步骤，执行一轮后，最右边应该存放着最大的数
  3. 进行一轮后，重复1，2步骤，但是不用比较最后一个数字
  4. 重复多轮，完成冒泡排序

python代码实现

list1 = [5, 4, 3, 2, 1]

def rank(list1):
    # 2个数据只要排一轮，3个数据只要排两轮,这里需要四轮
    for i in range(1, len(list1)):
        for j in range(len(list1) - i):
            if list1[j] > list1[j + 1]:
                temp = list1[j]
                list1[j] = list1[j + 1]
                list1[j + 1] = temp
    return list1


print(rank(list1))

代码改进

list1 = [5, 4, 3, 2, 1]

def rank(list1):
    for i in range(1, len(list1)):
        for j in range(len(list1) - i):
            if list1[j] > list1[j + 1]:
                list1[j], list1[j + 1] = list1[j + 1], list1[j]
    return list1


print(rank(list1))

2. 选择排序

算法步骤

  1. 首先在未排序的数组中，找到最小的那一个，和第一个交换位置
  2. 在剩下的数组中，找到最小的，排在第二位
  3. 排序好的，可以不用再查找最小的那个数了
  4. 重复，知道排序完成

python实现算法

list1 = [5, 4, 3, 2, 1]

def rank(list1):
    temp = 0
    for each in range(len(list1) - 1):
        for i in range(len(list1) - 1):
            if list1[i] > list1[i + 1]:
                #找到最小的那个，将他的下标存储在temp中，这里的temp是局部变量，所以在最开始让他赋值为0
                temp = i + 1
        new = list1[each]
        list1[each] = list1[temp]
        list1[temp] = new
    return list1

print(rank(list1))

代码改进

在上面的代码中，根本就不需要使用到交换顺序这个方法，此外，排序好的就不用再检查了,

`list1[each], list1[temp] = list1[temp], list1[each]`相当于
new = list1[each]
list1[each] = list1[temp]
list1[temp] = new

list1 = [5, 4, 7, 3, 2, 1]

def rank(list1):
    temp = 0
    for each in range(len(list1) - 1):
        for i in range(each, len(list1) - 1):
            if list1[i] > list1[i + 1]:
                #找到最小的那个，将他的下标存储在temp中，这里的temp是局部变量，所以在最开始让他赋值为0
                temp = i + 1
        list1[each], list1[temp] = list1[temp], list1[each]
        #等号号右边和左边不影响,只要保证temp=each，each=temp即可
    return list1

print(rank(list1))

插入排序法

算法步骤

  1. 现给出一个数组，将第0个看成是有序的，剩下的看成是未排序的
  2. 扫描未排序的第一个，将他插入到已排序中的合适的位置(和他之前那个比较，如果大小不对，则交换两者位置，每交换一次，temp不断向前移动一次，知道位置正确)
  3. 继续扫描未排序的第一个，重复步骤2，直到排序完成

python代码

list1 = [3, 1, 3, 22, 23, 1]

for i in range(1, len(list1)):
    temp = i
    for j in range(0, i):
        if list1[temp] < list1[i - 1 - j]:
        #这里需要temp存储交换后的位置，每交换一次减1，且每次只和之前的1个比较
            list1[temp - 1], list1[temp] = list1[temp], list1[temp - 1]
            temp -= 1

print(list1)
