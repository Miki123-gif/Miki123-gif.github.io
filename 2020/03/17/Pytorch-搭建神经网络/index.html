    Pytorch实现反向传播算法

    • pytorch好处就是能使用GPU进行计算，加快运算时间
    • numpy只能使用CPU进行运算，效率没它高

    import torch

# 创建一个张量并设置 requires_grad=True 用来追踪他的计算历史
x = torch.ones(2, 2, requires_grad=True)
print(x)
# tensor([[1., 1.],
#         [1., 1.]], requires_grad=True)


# 对张量进行计算操作，grad_fn已经被自动生成了。
y = x + 2
print(y)
# tensor([[3., 3.],
#         [3., 3.]], grad_fn=<AddBackward>)
print(y.grad_fn)
# <AddBackward object at 0x00000232535FD860>

# 对y进行一个乘法操作
z = y * y * 3
out = z.mean()

print(z)
# tensor([[27., 27.],
#         [27., 27.]], grad_fn=<MulBackward>)
print(out)
# tensor(27., grad_fn=<MeanBackward1>)

    • 也就是说，如果少了backward()，他就不会自动回带x的值，调用x.grad就会显示空
    • 当设置requird_grad=True时，就会对该张量贴一个标签，自动追踪这个张量的所有运算，所以即使复合运算也没关系

    • 当输入为多个的时候,先看看推导，始终有个偏置为1，他们共用一套系数

    import torch

data_x = [
    [1,2,1],
    [2,3,1]
]
target = [[2.1,3]]
data_w = [[3,4,5]]
stride = 0.01
x = torch.tensor(data_x).float()
y = torch.tensor(target).float()
w = torch.tensor(data_w).float()
w.requires_grad = True
count = 0
while True:
    count += 1
    out = torch.mm(x, w.t())
    temp = torch.sub(y, out.t())[0]
    loss_func = temp[0]**2 + temp[1]**2
    if loss_func.data < 0.00001:
        break
    loss_func.backward()
    w.data -= w.grad*stride
    w.grad *= 0
print(out)
print(out.data)

    tensor([[2.1026],
        [2.9983]], grad_fn=<MmBackward>)
tensor([[2.1026],
        [2.9983]])
    • 注意一定要用矩阵表示target = [[2,3]],不能用列表表示target = [2,3]
    • Tensor(data_x)，这样才能将矩阵转化成张量，在GPU中运算，转化后默认时浮点型
    • requires_grad=True，开启自动求导模式后，就会对变量w进行追踪，不论w进行了什么运算都能标记
    • torch.mm，进行张量的乘法
    • w.t()，张量的转置
    • torch.sub，torch中没有定义__sub__这个魔法方法，所以不能用-号操作
    • loss_func.backward()，在反向传播时，只能对损失函数进行反向传播，我们前面定义了w的自动求导，因为x为输入是固定的，不需要求导，定义后就会对损失函数中的w求导，并带x的值进去
    • w.grad *= 0 ##w.grad = torch.Tensor([[0,0,0]])，这一步是必须的，torch中的自动求导运算，每次循环都会进行梯度累加，所以每次循环必须要清零,为什么请看这里

    使用pytorch搭建网络

    • nn是neural network包，与神经网络有关
    import torch as t
import torch.nn as nn

x = t.tensor([[1, 2], [3, 4]]).float()
y = t.tensor([[3], [7]]).float()

my_net = nn.Sequential(
    nn.Linear(2, 3),
    nn.Sigmoid(),
    nn.Linear(3, 7),
    nn.Sigmoid(),
    nn.Linear(7, 1)
)
optimizer = t.optim.Adam(my_net.parameters(), lr=0.05)
loss_func = nn.MSELoss()

while True:
    out = my_net(x)
    loss = loss_func(out, y)
    if loss.data < 0.0001:
        break
    optimizer.zero_grad()
    loss.backward()
    optimizer.step()

print(my_net(x).data)

    对pytorch搭建网络的详细分析:

    • x = t.tensor([[1, 2], [3, 4]]).float()首先输入的数据一定要是浮点型

    my_net = nn.Sequential(
    nn.Linear(2, 3),
    nn.Sigmoid(),
    nn.Linear(3, 7),
    nn.Sigmoid(),
    nn.Linear(7, 1)
    )
    创建神经网络，要记得输入和输出相连接，一般最后一层不需要激励函数
    my_net实例化后，就有个parameters方法，生成权重
    • optimizer = t.optim.Adam(my_net.parameters(), lr=0.05)创建优化器实例，传入网络参数和学习率
    • loss_func = nn.MSELoss()调用损失函数方法，这里使用的是平均损失函数
    • out = my_net(x)，传入输入，loss = loss_func(out, y)，计算误差
    • optimizer.zero_grad()将优化器中的梯度清0
    • loss.backward()反向传播
    • optimizer.step()参数更新

    假如现在有新的数据，我们对其进行计算
