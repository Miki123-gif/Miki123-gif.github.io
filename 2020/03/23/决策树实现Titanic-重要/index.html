    决策树

    • 决策树对数据的要求不高，可以是数值型，或者字符串类型

    决策树如何找到关键特征

    1. 如何从数据的特征中找到最佳节点和最佳分支
    2. 如何停止决策树增长，防止过拟合

    过拟合和欠拟合

    1. 过拟合

    在训练集上表现很好，但是在测试集上表现很遭
    2. 欠拟合

    在训练中学习的就不是很好，在测试的时候依旧表现很差，这就是欠拟合

    泛化能力定义

    有时候，模型能够对训练集的数据学习的很好，当我们往训练集加入一些新的具有同样特征的数据时，模型得到的结果不尽人意，则该模型的泛化能力比较低

    sklearn中实现决策树(代码)

    决策树实现代码：

    1. 从sklearn中导入相关函数
    from sklearn import tree
    1. 对类进行实例化
    clf = tree.DecisionTreeclassifier()
    1. 用训练集训练模型
    clf = clf.fit(x_train, y_train)
    1. 返回预测的准确度
    result = clf.score(x_test, y_test)

    上面的几个函数，几乎适用于所有的sklearn算法

    模型改进

    交叉验证

    定义：
    交叉验证，就是重复的使用数据，把得到的样本数据进行切分，组合为不同的训练集和测试集。用训练集来训练模型，测试集来评估模型的好坏。在此基础上可以得到多组不同的训练集和测试集，某次训练集中的样本，在下次可能成为测试集中的样本

    交叉验证优点：

    前面我们对数据集的划分，都是随机的，决策树在训练时，会表现出有时候训练的很好，有时候训练的不好，是因为对特征的选取和数据的选取都是随机的，有些数据在学习时，表现很好，在测试时，得分也不错，我们就应该将该训练集挑选出来，为了增强模型的泛化能力，因此可以使用交叉验证的方法

    1. 交叉验证用于评估模型的预测性能，尤其是训练好的模型在新数据上的表现，可以在一定程度上减小过拟合。

    2. 可以从有限的数据中获取尽可能多的有效信息。

    重要参数说明：

    cross_validation.cross_val_score(estimator, X, y=None, scoring=None, cv=None)

estimator要传入的算法
x表示完整的数据集
y表示标签
cv表示折叠次数
scoring衡量模型的指标，改变这个，则输出结果也会改变

    from sklearn.model_selection import cross_val_score

# 如果不输出mean()，会输出列表
clf = DecisionTreeClassifier(random_state=0)
score = cross_val_score(clf, attribute, tag, cv=10).mean()

score

       0.76404494, 0.80898876, 0.79775281, 0.85393258, 0.80681818])

    参数调节 (重要)

    观察拟合程度

    在参数调节前，我们应当观察参数的拟合情况，观察是过拟合还是欠拟合

    • max_depth 参数是决策树中常调节的参数，它能有效防止过拟合，限制决策树的最大深度(因为决策树每多加一层，就需要很多新的特征，因此深度过大，容易造成过拟合)

    因此我们通过调节最大深度和画图，来观察过拟合情况

    import matplotlib.pyplot as plt

# 设置两个列表来装载拟合分数
te = []
tr = []
for i in range(10):
    # 每一次循环下都要重新实例化模型
    clf = DecisionTreeClassifier(random_state=0, max_depth=i + 1)
    # 前面我们对数据进行了分类，30%用于测试，70%用于训练
    clf = clf.fit(x_train, y_train)
    # 对30%的测试数据进行打分
    train_score = clf.score(x_test, y_test)
    # 交叉验证法，要传入完整的数据集，训练也会不断改变类对象的属性
    test_score = cross_val_score(clf, attribute, tag, cv=10).mean()
    te.append(test_score)
    tr.append(train_score)
# 画图观察结果
plt.figure()    
plt.plot(range(1,11), tr, color='blue', label='test')
plt.plot(range(1,11), te, color='black',label='train')
# 改变x轴
plt.xticks(range(1,11))
# 增加图例
plt.legend()
plt.show()

    • 当我们改变参数时，发现当层数为6的时候，两者拟合的都比较好
        clf = DecisionTreeClassifier(random_state=0
                                 , max_depth=i + 1
                                , criterion='entropy'
                                 )

    网格搜索

    from sklearn.model_selection import GridSearchCV
clf = DecisionTreeClassifier(random_state=0)
parameters = {
    'criterion':['gini','entropy']
    ,'splitter':['best','random']
    ,"max_depth":range(3,9)
}

# 网格搜索本身就会进行交叉验证
gs = GridSearchCV(clf, parameters, cv=10)

# 网格搜索就相当于新的算法，可以进行训练
gs = gs.fit(x_train, y_train)

print(gs.best_params_)# 查看最佳参数
print(gs.best_score_)# 查看跑的最好分数

{'criterion': 'entropy', 'max_depth': 3, 'splitter': 'best'}
