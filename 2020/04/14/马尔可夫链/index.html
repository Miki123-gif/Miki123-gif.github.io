    马尔可夫问题：**

    我们用马尔可夫链，来描述马尔可夫随机过程

    马尔可夫过程，是随机过程的核心，可以看看知乎上随机过程讲解，详细讲述了概率，通俗易懂，很有帮助，学习一门知识，不应该一上来就是讲数学，但最后应该回归数学，用数学的语言去描述

    概率其实是我们根据已有的知识，对将来的事进行预测，这个事件能重复多次实验，如抛一枚硬币，我们知道得到正面的概率是1/2

    1. 是因为我们知道硬币有两面，已经具有这样的知识
    2. 是因为我们能进行多次实验，假如这个事件只能发生一次，又何来的1/2呢
    • 假如现代人生小孩，只要生了男孩，则不会生小孩了，如果生的是女孩，则会继续生下去，直到生男孩为止，这样以后会造成男女比例失调吗？

    生育问题是典型的稳态马尔科夫过程，下一次生育不受上一次生育的影响。 根据马氏过程的特性，你知道历史无需考虑历史路径， 最终的平衡概率只取决于每一步的概率。所以无论你怎么玩，不论是你拼命想生男孩还是女孩，都无法影响人口比例。

    import random


class Family:
    def __init__(self):
        self.male = 1000
        self.female = 1000

    def birth(self):
        result = random.choice([0, 1])
        if result == 1:
            self.male += 1
        else:
            self.female += 1
            self.birth() # 利用递归实现

    def proportion(self):
        for i in range(1000000):# 设置家庭数量
            self.birth()
        print(f'{self.male/(self.male + self.female)}')


f = Family()
f.proportion() # 0.49965832555061

    计算机仿真结果说明，这样是不造成男女比例失调的，结果总是在0.5上下波动，这其实就是马尔可夫随机过程

    1. 马尔可夫随机过程是什么？

    一个事物的将来，只于其现在的状态有关，而不依赖其过去，这里有两份资料

    • 股市的马尔可夫模型，有代码讲解
    • 马尔可夫的状态矩阵，有例子说明

    假如一个模型符合马尔可夫过程，只要他的状态矩阵确定了，我们可以根据其现在的状态，推测出其将来的状态是多少

    假如现在有0，1，2三个状态，0状态可以保持不变，也可以转换到1，2状态

    00表示当前状态0变换到将来状态0的概率是0.1，这就是转移矩阵

         0    1    2
0   [0.1,0.2,0.3]
1   [0.1,0.2,0.3]
2   [0.6,0.5,0.4]

    假如之前的状态0，1，2无所谓，我们知道，他转移到下一个状态的概率为[0.5,0.6,0.4],这个行向量就被称做初始状态，当我们知道这个概率时，就可以计算将来的所有状态概率，只需要用这个行向量乘以转移矩阵p即可，得到一个新的行向量，这个结果再乘以矩阵p，就可以得到以后所有的概率，具体可以看上面的第二个链接

    总结：

    如果一个过程是markov过程，这个过程就得到了神简化，你只需要知道第n步是如何与第n-1步相关的，一般由一组条件概率表述，就可以求得整个过程。一个巨大的随机过程，其内核仅仅是这样一组条件概率，而知道了这组条件概率，就可以衍生整个过程。

    有趣的现象：

    当转移矩阵设定好并且不变时，无论初始的状态设置为多少，只要转移矩阵不变，那么计算无数多次后，行向量会逐渐收敛

    马尔科夫链的收敛条件：

    1. 可能的转移状态是有限的
    2. 状态转移矩阵，不发生变化(重要)
    3. 状态之间，能够相互转变，也就是要能够形成一个状态矩阵
